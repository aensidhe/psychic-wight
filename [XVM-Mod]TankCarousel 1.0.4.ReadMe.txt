XVM-Mod - TankCarousel(Tier + Type filters)

Автор: STL1te
Версия XVM: 5.0.0+

Описание: Мод для XVM, добавляющий два новых фильтра карусели танков. Первый фильтр по уровню, второй по типу (подробности ниже). 
Рядом со стандартными фильтрами появляются два новых. Каждый из них представляет из себя выпадающее меню с пунктами в виде изображения с чекбоксом. Можно выбрать несколько пунктов одновременно.
В фильтре по уровню находится список уровней (от 1 до 10).
В фильтре по типу находится список типов:
- Элитный танк
- Премиум танк
- Обычный танк
- Танк, на котором не "сбили звезду".
ОБРАТИТЕ ВНИМАНИЕ: ВСЕ премиумные танки являются элитными и следовательно, чтобы выбрать в фильтре только премиумные танки нужно выбрать пункты (поставить галочки) на "Элитный танк" и "Премиум танк".

Установка: Скопировать папку "res_mods" из архива в корень папки с игрой (напр. "C:/Games/World_of_Tanks") и подтвердить замену (нужные файлы не заменятся).
